# Responsividade Bikcraft

Estudo prático focado em responsividade utilizando CSS.

##

## Sobre

Este projeto foi desenvolvido com foco na adaptação de layouts para diferentes tamanhos de tela.

A proposta foi aplicar conceitos de responsividade em uma interface baseada no projeto Bikcraft.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* uso de media queries
* adaptação de layouts para dispositivos móveis
* ajustes de tipografia e espaçamento
* construção de interfaces responsivas

##

## Conteúdos

### Responsividade

* uso de media queries (@media)
* adaptação de layout com flex e grid
* ajustes visuais para diferentes resoluções
* tratamento de imagens responsivas
