# Estudos

Diretório com materiais e exercícios desenvolvidos durante os estudos na Origamid.

##

## Sobre

Este diretório reúne exercícios e exemplos criados ao longo das aulas, utilizados como reforço e referência prática dos conteúdos estudados.

Os materiais são organizados por tema, facilitando a consulta e revisão.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* prática de conceitos de HTML e CSS
* experimentação com JavaScript
* estudo de layout e responsividade
* organização de código
* reforço de conteúdos teóricos
