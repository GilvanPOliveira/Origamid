# Pseudo-classes

Estudo prático focado no uso de pseudo-classes com CSS.

##

## Sobre

Este projeto foi desenvolvido com foco na aplicação de estados e seletores condicionais no CSS.

A proposta foi explorar como interações e condições podem alterar o comportamento visual dos elementos.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* uso de pseudo-classes no CSS
* criação de estados interativos
* seleção de elementos por condição
* construção de interfaces mais dinâmicas

##

## Conteúdos

### Pseudo-classes

* uso de :hover, :focus e :active
* seleção com :first-child, :last-child e :nth-child()
* estados de formulário (:checked, :disabled)
* aplicação de seletores condicionais
