# Radio Button Customizado

Estudo prático focado na criação e estilização de radio buttons com CSS.

##

## Sobre

Este projeto foi desenvolvido com foco na personalização de inputs do tipo radio.

A proposta foi criar uma interface visual customizada mantendo o comportamento nativo e a acessibilidade.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* estruturação de formulários com HTML
* estilização de inputs com CSS
* personalização de componentes nativos
* manutenção de usabilidade e acessibilidade

##

## Conteúdos

### Formulários

* uso de input[type="radio"] com label
* controle de estados como :checked e :focus
* estilização de radio buttons customizados
* preservação do comportamento padrão do navegador

##

## Estrutura do Projeto

```text
index.html
style.css
```

##

## Execução Local

* abrir o arquivo `index.html` no navegador  
* ou utilizar a extensão Live Server no VS Code
