# Cinema

Estudo prático focado em layout, tipografia e organização de conteúdo com HTML e CSS.

##

## Sobre

Este projeto foi desenvolvido com base em um tema de cinema para praticar a construção de interfaces.

A proposta foi aplicar conceitos de estruturação, tipografia e organização visual de conteúdo.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* estruturação de páginas com HTML
* organização de seções e listas
* aplicação de tipografia
* construção de layouts simples com CSS

##

## Conteúdos

### Layout e Tipografia

* estrutura de seções e organização de conteúdo
* uso de tipografia (tamanho, peso e espaçamento)
* construção de cards e blocos de conteúdo
* aplicação de padrões de interface com CSS
