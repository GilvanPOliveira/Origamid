# CSS Grid (Exemplos)

Estudo prático focado na construção de layouts utilizando CSS Grid.

##

## Sobre

Este projeto foi desenvolvido com foco na criação de layouts mais estruturados utilizando CSS Grid.

A proposta foi explorar diferentes formas de organização de elementos em grade.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* construção de layouts com CSS Grid
* organização de conteúdo em áreas
* controle de alinhamento e espaçamento
* adaptação de layout para diferentes telas

##

## Conteúdos

### CSS Grid

* criação de layouts em grid
* uso de áreas (quando aplicável)
* controle de colunas e linhas
* ajustes de alinhamento e distribuição
* responsividade com grid
