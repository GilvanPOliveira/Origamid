# Modo Claro/Escuro

Estudo prático focado na implementação de temas claro e escuro.

##

## Sobre

Este projeto foi desenvolvido com foco na criação de temas visuais alternáveis.

A proposta foi aplicar conceitos de estilização com CSS e, quando necessário, lógica com JavaScript para alternância e persistência de tema.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* criação de temas com CSS
* uso de variáveis para controle de cores
* alternância de tema via interação
* persistência de preferências do usuário

##

## Conteúdos

### Temas

* implementação de modo claro e escuro
* uso de variáveis CSS para controle de tema
* alternância de tema com classes e/ou JavaScript
* persistência com localStorage (quando aplicável)
* boas práticas de contraste e legibilidade
