# CSS Variáveis

Estudo prático focado no uso de CSS Custom Properties (variáveis).

##

## Sobre

Este projeto foi desenvolvido com foco na utilização de variáveis no CSS para melhorar a organização e reutilização de estilos.

A proposta foi aplicar conceitos de tema, espaçamento e padronização visual utilizando CSS Custom Properties.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* uso de variáveis no CSS
* organização de estilos reutilizáveis
* manutenção e escalabilidade de código
* criação de temas e padronizações visuais

##

## Conteúdos

### CSS Custom Properties

* declaração de variáveis com :root
* reutilização com var(--nome)
* escopo de variáveis (global e local)
* uso de fallback e manutenção de estilos
