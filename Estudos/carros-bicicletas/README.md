# Carros e Bicicletas

Estudo prático focado na estruturação de layout com HTML e CSS.

##

## Sobre

Este projeto foi desenvolvido com foco na organização visual de conteúdo utilizando um exemplo simples.

A proposta foi aplicar conceitos básicos de estilização e estruturação com HTML e CSS.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* uso de estrutura semântica no HTML
* organização de layout com CSS
* aplicação de espaçamentos e alinhamentos
* construção de hierarquia visual

##

## Conteúdos

### Layout

* estruturação semântica de conteúdo
* uso de margin e padding
* aplicação de cores e bordas
* organização de blocos e alinhamento

##

## Estrutura do Projeto

```text
index.html
style.css
```

##

## Execução Local

* abrir o arquivo `index.html` no navegador  
* ou utilizar a extensão Live Server no VS Code
