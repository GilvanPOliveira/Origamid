# Tipografia

Estudo prático focado em tipografia utilizando CSS.

##

## Sobre

Este projeto foi desenvolvido com foco no estudo de tipografia no CSS.

A proposta foi explorar conceitos de legibilidade, hierarquia visual e organização de texto em interfaces web.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* uso de propriedades tipográficas
* organização de hierarquia textual
* melhoria de legibilidade
* aplicação de boas práticas de design

##

## Conteúdos

### Tipografia

* uso de font-family, font-size e font-weight
* controle de line-height e espaçamento
* definição de hierarquia entre títulos e parágrafos
* aplicação de boas práticas de leitura

##

## Estrutura do Projeto

```text
index.html
style.css
```

## Execução Local
