# Carrossel de Imagens

Estudo prático focado na criação de um carrossel de imagens com JavaScript.

##

## Sobre

Este projeto foi desenvolvido com foco na manipulação de elementos do DOM utilizando JavaScript.

A proposta foi criar um carrossel de imagens com navegação entre itens e atualização dinâmica da interface.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* manipulação do DOM com JavaScript
* controle de estados e navegação
* uso de eventos para interação
* organização de lógica em funções reutilizáveis

##

## Conteúdos

### JavaScript

* lógica de navegação (próximo e anterior)
* manipulação de classes e estados
* eventos de interação
* atualização dinâmica do DOM

##

## Estrutura do Projeto

```text
img/
index.html
script.js
style.css
```

## Execução Local
