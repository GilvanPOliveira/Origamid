# CSS Utilitário

Estudo prático focado na criação e uso de classes utilitárias com CSS.

##

## Sobre

Este projeto foi desenvolvido com foco na construção de estilos reutilizáveis.

A proposta foi aplicar conceitos de CSS utilitário para acelerar o desenvolvimento e manter consistência visual.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* criação de classes utilitárias
* reutilização de estilos
* padronização de layout e tipografia
* organização escalável de CSS

##

## Conteúdos

### CSS Utilitário

* organização de classes reutilizáveis
* uso de utilitários para espaçamento e tipografia
* padronização de cores e estilos
* estrutura escalável de CSS
