# Pseudo-elementos

Estudo prático focado no uso de pseudo-elementos com CSS.

##

## Sobre

Este projeto foi desenvolvido com foco na criação de elementos visuais utilizando pseudo-elementos.

A proposta foi aplicar técnicas de estilização sem a necessidade de adicionar elementos extras no HTML.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* uso de pseudo-elementos no CSS
* criação de elementos decorativos
* organização visual sem aumento da estrutura HTML
* aplicação de boas práticas de estilização

##

## Conteúdos

### Pseudo-elementos

* uso de ::before e ::after
* utilização da propriedade content
* posicionamento e estilização de elementos
* criação de detalhes visuais com CSS puro

##
