# Animais Fantásticos

Aplicação web desenvolvida para prática de JavaScript moderno (ES6).

##

## Sobre

Este projeto foi desenvolvido com foco na aplicação de conceitos do JavaScript moderno.

A proposta foi construir uma interface interativa utilizando manipulação de DOM, eventos e organização de código.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* uso de JavaScript ES6
* manipulação de elementos no DOM
* controle de eventos e interações
* organização de código em módulos e funções

##

## Funcionalidades

### Interface

* navegação entre seções
* interação dinâmica com elementos

##

### Interatividade

* manipulação de classes e estados
* eventos de clique e navegação
* atualização dinâmica do conteúdo
