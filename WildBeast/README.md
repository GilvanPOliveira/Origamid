# WildBeast

Aplicação web desenvolvida para prática de CSS Grid Layout.

##

## Sobre

Este projeto foi desenvolvido com foco no aprendizado e aplicação de layouts utilizando CSS Grid.

A proposta foi construir uma interface organizada utilizando o sistema de grid para posicionamento e distribuição de elementos.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* estruturação de páginas com HTML
* estilização com CSS
* uso de CSS Grid para layout
* organização de elementos na interface
* construção de layouts responsivos

##

## Funcionalidades

### Interface

* layout estruturado com grid
* organização visual de conteúdos

##

### Layout

* posicionamento de elementos com CSS Grid
* controle de áreas e distribuição de conteúdo

##

### Responsividade

* adaptação para diferentes tamanhos de tela
